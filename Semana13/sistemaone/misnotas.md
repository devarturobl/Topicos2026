# Bienvenidos a este proyecto con Laravel y Filamente
## Comandos Basicos
### Comando para ejecutar el servidor
    php artisan serve
### Comando para actualizar o generar la base de datos
    php artisan migrate
### Comando para crear un usuario en filamente
    php artisan make:filament-user
### Comando para crear un resource
    php artisan make:filament-resource NOMBRE --generate
NOTA: para generar un resource se necesita tener un modelo y una migration (tabla)
### Comando para actualizar Tablas de BD y agregar seeds
    php artisan migrate:fresh --seed
